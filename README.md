# Uurssaf_project_microentreprise
Project focused on self-employed workers in France, particularly those in micro-enterprises. The aim of this project is to highlight the most attractive activities, but also the most “profitable” ones (estimate). Highlight a few indicators that can be used to identify the most successful activities and those that are less so.

Projet axé sur les travailleurs indépendants en France, en particulier ceux des micro-entreprises. L'objectif de ce projet est de mettre en avant les activités les plus attractives, mais aussi les plus « rentables » (estimation). Mettre en évidence quelques indicateurs permettant d'identifier les activités les plus performantes et celles qui le sont moins.

Métriques principales = Chiffre d'affaires (CA) hors taxe; Revenu avant impot; Nombre de travailleurs autonomes;

Pour chacune de ces 3 metriques = Taux d'évolution globale; Taux d'évolution (Taux d'évolution pondéré si plusieurs années observées);

% revenue sur CA;

La modélisation des données s'effectue à travers SQL Server et la visualisation à l'aide de Power BI
